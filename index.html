Features

Markdown-first

Slides separated by ---. Headings set the layout. Write like you always have. Kova produces clean slides from standard Markdown.

Auto layouts

Title, section, two-column, quote, grid, and full-bleed layouts — detected automatically from your content structure.

Mermaid diagrams

Pie, bar, and line charts plus flowcharts — rendered inline from fenced code blocks. No image exports needed.

Syntax highlighting

Fenced code blocks rendered with highlight.js — 190+ languages.

Keyboard-driven

Every action has a shortcut — fully remappable via the inspector or ~/.kova/keybindings.yaml.

And much more

Presenter view, speaker notes, PowerPoint export, drag-and-drop images, progress bars, YouTube embeds, QR polls, and more.
